I have structure this project in multiple folder to manage all the things
for Ex.
Folder

/services ( All module have separate service All the logic will be there not in controller  )
/controller ( All module have separate controller )
/middleware ( Middleware )
/models ( All module have separate controller )
/validation ( All module have separate validation folder and and there rules is present there )
/util (  All the utility function in these folder)


To run this folder You have to define require environment variable refer .env.example

And to start the product 
You have to use below command  
npm run start
